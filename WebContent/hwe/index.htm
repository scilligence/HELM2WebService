HELM Web Editor

(Release 2(pre), May 21, 2017)

HELM Web Editor is an editor to draw and display macro molecules, built with 100% javascript, depending on JSDraw.Lite, dojo toolkit and pako.js.

Examples

  • HELM Web Editor App
  • HELM Monomer Manager
  • HELM RuleSet Manager

  • Web Service Documentation (Swagger)

Download Binary

1. Download HELM Web Service for Tomcat

helm2-webservice-2.0.0.war

To deploy it on Tomcat, rename the war file to WebService.war, then copy WebService.war file to Tomcat webapps folder (e.g. C:\Program Files\Apache Software Foundation\Tomcat 9.0\webapps)

The editor can be loaded in browser by using URL: http://localhost:8008/WebService/hwe

2. For IIS web server, please contact Scilligence.

Attention for Deployment

  • This war build supports Tomcat 8 and later version, but not support Tomcat 7
  • In order to be able to open files in HELM Web Editor, this line should be manually configured in web.xml:
    <location>c:\temp</location>
